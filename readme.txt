## ConsoleGame_2048
Clone of the famous game 2048, made to work in the console.
https://en.wikipedia.org/wiki/2048_(video_game)

## ConsoleGame_Battleship
Console version of the board game "Battleship."
The current version allows the player to try to destroy ships generated by the computer. No multiplayer mode.
https://en.wikipedia.org/wiki/Battleship_(game)

## ConsoleGame_CoffeeMath
The game generates 20 simple mathematical expressions, including addition, subtraction, and multiplication. Solve them in the shortest amount of time.
Every wrong answer will add 5 seconds to your final time.

## ConsoleGame_ConnectFour
Console version of the Connect Four board game. Use the numbers 1 to 7 to place the token on the board. The first of two players to connect 4 tokens wins.
No computer player yet.
https://en.wikipedia.org/wiki/Connect_Four

## ConsoleGame_Mastermind
Console version of the board game "Mastermind." The goal is to break the code generated by the computer. Instead of colors, the game uses numbers.
https://en.wikipedia.org/wiki/Mastermind_(board_game)

## ConsoleGame_Zekohop
Console version of the game "Jump In" by Smart Games.
https://www.smartgames.eu/uk/one-player-games/jump

## SoftUni_DragonFight
Console game where you equip a warrior to fight a dragon. Developed as an assignment in the SoftUni school for programming.
